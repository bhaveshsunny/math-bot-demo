# math-bot-demo

## Instructions (for HMR live server on webpack)

```
git clone https://github.com/bhaveshsunny/math-bot-demo
npm install
npm run watch
```

access on http://0.0.0.0:8080

## Or (for static server [production mode])

```
git clone https://github.com/bhaveshsunny/math-bot-demo
npm install
npm run build
npm run start
```
access on http://0.0.0.0:8080
